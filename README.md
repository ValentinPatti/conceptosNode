# Conceptos de NodeJS

Este proyecto es una practica de los primeros conceptos de node.js, donde implementaremos middlewares, endpoints y fue construido con pnpm como gestor de paquetes.

## Demo del backend

Podes ver una demo del proyecto en produccion [aqui](https://conceptos-node-five.vercel.app/)

## Librerias utilizadas

- Node js v24.4 o cualquier version posterior a la 22.
- ExpressJS
- Cors
- Morgan

## Instalacion y configuracion del proyecto

1- Clonar el repositorio 

`git clone https://github.com/ValentinPatti/conceptosNode.git`

2- Instalar las dependencias

`pnpm install`

3- Iniciar aplicacion: 

```bash
#comando de produccion 
pnpm start

#comando de desarrollo
pnpm run dev
```
